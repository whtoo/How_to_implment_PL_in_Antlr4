# 以`CFG`和`地址化`为中心的编译构造实作

## 测试样例源码

``` CPP

int dec1(int x) {
    return x - 1;
}

int main() {
    int i = 10;
    while(i > 0) {
        if (i > 5) {
            print(i);
            if(i == 7) {
                return 7;
            }
        }

        print("break");

        i = dec1(i);
    }

   return 0;
}



```

### 线性化IR
我们的IR本质上是tree模式的，这样一来我们的线性化实际上延迟到了指令生成时。
但是，这样一来我们就无法进行活性分析和很多与TAC表示相关的分析（或者是我没找到
直接对栈代码做分析的例子）。因此，我要对ep20的输出code过程进行线性改造。
