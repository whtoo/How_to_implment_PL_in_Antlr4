## 技术成果

### 语法规则扩展
```antlr
// 新增运算符
expr o=('*'|'/'|'%') expr    #exprBinary
expr o='&&' expr #exprLogicalAnd

// 数组语法
varDecl: primaryType ID ('[' expr ']')? ('=' (expr | arrayInitializer))? ';'
expr: expr '[' expr ']' #exprArrayAccess
arrayInitializer: '{' expr (',' expr)* '}'

// 函数参数支持数组
formalParameter: primaryType ID ('[' ']')?
```
