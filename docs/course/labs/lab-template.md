## 实验报告要求

### 报告格式
1. **封面**：实验名称、学号、姓名、日期
2. **摘要**：实验目的、内容、结果
3. **正文**：实验过程、结果分析
4. **结论**：实验总结、心得体会
5. **附录**：关键代码、运行结果

### 内容要求
1. **实验过程**：详细记录实验步骤
2. **结果分析**：分析实验结果
3. **问题讨论**：讨论遇到的问题和解决方法
4. **创新点**：如有创新，详细说明

### 提交要求
1. **电子版**：PDF格式，文件名：学号-姓名-实验编号.pdf
2. **代码**：Git提交记录或压缩包
3. **截止时间**：[截止时间]

## 评分标准

### 代码质量（40%）
- **正确性**：20%（功能实现正确）
- **规范性**：10%（代码规范、注释）
- **效率**：10%（算法效率、资源使用）

### 实验报告（40%）
- **完整性**：15%（内容完整、格式规范）
- **分析深度**：15%（分析深入、见解独到）
- **规范性**：10%（报告规范、语言流畅）

### 实验过程（20%）
- **完成度**：10%（按时完成所有任务）
- **创新性**：5%（有创新思考或实现）
- **团队协作**：5%（如有团队协作）

### 总分计算
- 优秀：90-100分
- 良好：80-89分
- 中等：70-79分
- 及格：60-69分
- 不及格：<60分
